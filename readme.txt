Provides a widget that allows you to easily share your social profiles on your website.

== Description ==

The BossSocial plugin allows you to link of the most popular social networking sites. The plugin comes with a custom widget allowing you to place social icons in any widget location or sidebar or place them in posts and pages with the BossSocial shortcode.

== Installation ==

1. Upload the /boss-social/ folder to the /wp-content/plugins/ directory.
1. Activate the BossSocial plugin through the "Plugins" menu in WordPress.
1. Configure your settings in the new "BossSocial" menu which appears.
1. Display your icons using the custom widget or shortcode.

== Frequently Asked Questions ==

= To use BossSocial in your posts and pages you can use the shortcode:  =

[boss_social_icon type="" url=""] 

= You can use "type" parameter to select social icon eg:  =

[boss_social_icon type="facebook" url=""]
